Reducing size of a PDF file

Install ghostscript and run the following command

gs -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -dCompatibilityLevel=1.4 -dPDFSETTINGS=/screen -sOutputFile=new_file.pdf original_file.pdf

Options for -dPDFSettings:

/screen selects low-resolution output, and the lowest file-size.
/ebook selects medium-resolution output, with a medium file-size.
/printer and /prepress are both the high-resolution options, which is mainly used for printing PDFs.

Video Format conversion

Converting video files from one format to another
ffmpeg -i movie.mov -f mp4 -vcodec copy -acodec copy output.mp4
Remove "-vcodec copy -acodec copy" in case of errors

Converting video files from one format to another and compress
ffmpeg -i input.mp4 -vcodec libx264 -crf 20 output.mp4
Vary the crf parameter from 18 to 24 - the lower, the higher the bitrate
