• alignas specifier

    • Aligning objects on a specified byte boundary.

    cppreference

    struct alignas (16) Employee {
      uint8_t age;
      double salary;
      std::string name;
    };


  • override specifier

    • - Specifies that a virtual function is overriding another virtual function. Avoids accidental function overloading if the datatypes of parameters are changed by mistake. For instance, using "double" for the overriding function parameter instead of "const double". Compiler throws an error when function signatures mismatch.
    - Only for virtual functions

    cppreference

    struct Engineer : public Employee {
      void computeLeavesTaken () override ;
    };

    pair operator , (Place p1, Place p2)
